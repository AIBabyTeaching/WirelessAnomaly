# Wireless Anomaly Detection

This project provides the reference implementation for the paper
["AI-Driven Anomaly Detection for Wireless Networks"](https://doi.org/10.17632/p4n85smvms.1).
It offers a modular, reproducible pipeline for analysing wireless propagation
scenarios and detecting anomalies using kernel principal component analysis and
Isolation Forests.

The repository ships with a command-line interface, rich documentation and
utilities to reproduce figures and tables from the accompanying publication.
