# Data

The Mendeley dataset [10.17632/p4n85smvms.1](https://doi.org/10.17632/p4n85smvms.1)
contains reduced KPCA components and labels. Place downloaded files under
`data/external/` and point the configuration file to the desired path. Raw
features can be stored in `data/raw/` for future end-to-end experiments.
