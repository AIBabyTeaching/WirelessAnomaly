# Wireless Anomaly Detection

[![CI](https://img.shields.io/badge/CI-passing-green)](#)
[![License](https://img.shields.io/badge/license-MIT-blue)](LICENSE)

Prototype framework for anomaly detection using KPCA and Isolation Forest.

## Quickstart
```bash
pip install -e .
wireless-anom run reduced --config src/wireless_anom/config/defaults.yaml
```

Dataset available via DOI [10.17632/p4n85smvms.1](https://doi.org/10.17632/p4n85smvms.1).
